# NodeJS 读写 Excel

## Excel概念

>workbook 对象，指的是整份 Excel 文档。我们在使用 js-xlsx 读取 Excel 文档之后就会获得 workbook 对象。
>
>worksheet 对象，指的是 Excel 文档中的表。我们知道一份 Excel 文档中可以包含很多张表，而每张表对应的就是 worksheet 对象。
>
>cell 对象，指的就是 worksheet 中的单元格，一个单元格就是一个 cell 对象。
